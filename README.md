# Weather Pipeline

A Python ETL pipeline that fetches real-time weather data from the wttr.in API, transforms the data into a structured format, and triggers alerts based on configurable temperature thresholds.

## Features
- Extracts live weather data from an external API
- Processes and structures JSON responses
- Generates temperature alerts
- Handles API and data parsing errors
- Demonstrates the ETL (Extract, Transform, Load) workflow

## Technologies Used
- Python 3
- Requests
- JSON
